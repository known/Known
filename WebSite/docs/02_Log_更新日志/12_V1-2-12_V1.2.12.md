**4. 富文本组件**

- 基于wangEditor.js实现
- 编辑器配置通过Option参数设置

```csharp
protected override void BuildRenderTree(RenderTreeBuilder builder)
{
    builder.Field<RichText>("内容：", "RichText1")
           .Set(f => f.Option, new
           {
               Height = 200,
               Placeholder = "请输入通知内容"
           })
           .Build();
}
```
