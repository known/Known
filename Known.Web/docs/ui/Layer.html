弹出层 Layer

选项

option = {
    width: 数值，默认500px,
    height: 数值，默认300px,
    title: 字符,
    icon: 字符,
    url: Url,
    partial: 布尔，url是否是部件,
    content: Html字符,
    footer: Html字符,
    buttons: [
        { text: 按钮名称, handler: 按钮方法 }
    ],
    success: function () { //弹出后回调方法 }
}

方法

名称 参数 返回值 描述 示例
page option：选项对象，参见下文。 对话框对象，参见下文。 加载page页面。 Layer.page(option); 测试页面 测试部件
open option：选项对象，参见下文。 对话框对象，参见下文。 弹出模态对话框。 Layer.open(option); 无标题 测试内容 测试页面 测试部件
loading message：加载信息。 对话框对象对象，参见下文。 弹出加载中。 Layer.loading(message); 测试 
var dlg = Layer.loading('提交中......');
$.post(url, data, function(res) {
    dlg.close();
});
tips message：提示信息。 弹出提示，3秒后自动关闭。 Layer.tips(message); 测试
alert message：提示信息；
callback：确定后回调函数。 		弹出信息提示框。 Layer.alert(message, callback); 测试
confirm message：提示信息；
callback：确定后回调函数。 		弹出确认提示框。 Layer.confirm(message, callback); 测试

对话框对象

成员 描述
index 对话框实例的序号。
layer 对话框实例的jquery对象。
close 对话框关闭方法。
